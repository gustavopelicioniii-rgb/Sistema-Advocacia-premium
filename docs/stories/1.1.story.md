## 🏗️ Arquitetura

```
┌─────────────────┐
│   Frontend      │
│  (React App)    │
└────────┬────────┘
         │ POST /functions/v1/importar-processos
         │ { oab_estado, oab_numero }
         │
┌────────▼────────────────────────┐
│  Supabase Edge Function         │
│  importar-processos             │
│                                  │
│  - Lê ESCAVADOR_TOKEN do env     │
│  - Chama API Escavador           │
│  - Salva processos na tabela     │
└────────┬────────────────────────┘
         │
┌────────▼──────────────────┐     ┌──────────────────┐
│  Supabase Database        │     │  API Escavador   │
│  Tabela: processos        │◄────┤  (External)      │
│  - numero_processo        │     └──────────────────┘
│  - tribunal               │
│  - ultima_movimentacao    │
│  - last_checked_at        │
└───────────────────────────┘

┌──────────────────────┐
│ Supabase Scheduler   │
│ verificar-           │
│ movimentacoes        │ (1x/dia)
│ ↓                    │
│ Edge Function        │
└──────────────────────┘
```
